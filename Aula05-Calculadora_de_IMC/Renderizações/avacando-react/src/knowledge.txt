.replace() - Ele é utilizado para substituir uma parte específica de uma string por outra.

Ex:
const texto = "Eu amo JavaScript";
const novoTexto = texto.replace("JavaScript", "React");
// Resultado: "Eu amo React"


Expressões regulares são padrões utilizados para selecionar combinações de caracteres em uma string. 
Em JavaScript, expressões regulares também são objetos. 
Elas podem ser utilizadas com os métodos exec e test do objeto RegExp, e com os métodos match, 
replace,search, e split do objeto String. Este capítulo descreve o uso de expressões regulares em 
JavaScript.

ex: 
text.replace(/[^0-9,]/g,"");
